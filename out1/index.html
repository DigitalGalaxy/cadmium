Cadmium Demo

Welcome to the Cadmium demonstration! Hopefully this demonstration will help you understand the different features that Cadmium offers and how it can be used to publish static content in an efficient manner.

Cadmium is not a CMS

Although Cadmium does deal with content, it does not provide the features that you would associate with a modern CMS. Cadmium does not provide WYSIWYG editors for page content or templating engines for page layout. It leaves those tasks to other applications.

Then What is Cadmium, Technically?

Cadmium is a Content Versioning and Distribution System. It is intended to be tied with applications that render static content, like DocPad or Octopress. This setup has several advantages:

  • It moves content rendering out of the HTTP request/response cycle, improving page load times.
  • It reduces the number of moving parts in production, reducing the risk of failure.
  • It allows Web Developers to have more control over the content creation process.
