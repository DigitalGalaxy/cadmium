Cloning Content

During the development of an interactive website, it needs to move through different environments on its way to production. Cadmium is designed to ease this process by allowing users to clone content from one website to another.

The Clone Command

The Cadmium CLI Clone command performs the content distribution feature of Cadmium. This command takes a source website and a destination website as arguments. After the command completes, the content of the target website will be the same as source website. Here is the format of the command:

> cadmium clone -m <message> <source_site> <target_site>

This command is used recursively to move content from the development site to production. To understand this better, let's look at an example production push.
