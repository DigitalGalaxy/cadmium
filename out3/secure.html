Secure Pages

Cadmium provides support for http/https redirection for text/html pages. Cadmium does not redirect other content types, so assets can be served under either protocol.

Making a Page Secure

The /META-INF/secure.json file is used to configure pages that are secure. Pages that match on of the path regular expressions in this file will only be served under the https protocol. All other text/html resources will only be served under the http protocol. This config file will make any path starting with /secure be served under https:

[
  {
    "url":"/secure.*"
  }
]
